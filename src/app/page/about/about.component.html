Features of this app
  • Frontend: Angular
  • Backend: Spring Boot
  • Database: Uses IndexedDB API when offline,
    MySql when connected with backend
  • Perk: Item description supports Markdown formatting
  • Search: Prefix matching when offline,
    Fast MySQL Fulltext
  • Filtering: Advanced Grouping with tags
    (eg - "searchQuery !T: +tag1, -tag2") '+':include, '-':exclude
  • Backup: Download your notes as a json file and load notes from it as backup
