Features of this app
  • Frontend: Angular
  • Database: Uses IndexedDB API when offline,
  • Markdown: Item description supports Markdown formatting
  • Search: Prefix matching normally or full text with !F:, Cumulative with !ALL: or exact match otherwise
  • Sorting: Sort search by creationTimestamp with !OLD: or updationTimestamp with !LAT:
  • Filtering:Advanced Grouping with tags
    !T: with prefix '+':include, '-':exclude
    • !ALL: any that match (default - that matches all)
    • !F: full text search (default - subject prefix)
    • !T: tag based search with +,- modifier prefix
      Keyboard shortcuts - " Ctrl + Shift + { " -> home and " Ctrl + Shift + } " -> editor
  • Custom Fields: Add Custom fields to the todo item [very handy for visualization] with your json schema,
    this uses angular's reactiveforms and you can add validations
  • Backup: Download your notes as a json file and load notes from it as backup
